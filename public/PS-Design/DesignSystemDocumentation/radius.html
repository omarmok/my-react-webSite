Accessibility

  • Use larger radius for touch-friendly surfaces to communicate affordance.
  • Keep radius choices consistent so components remain recognizable.

Usage Guidelines

Containers

  • Use $radius-md to $radius-lg for cards and panels.
  • Apply $Radius-100 for pill or full-round shapes.

Controls

  • Match button radius to the component specification.
  • Use aliases when the component references Radius tokens.

Do and Do Not

Do

  • Use tokens instead of custom radius values.
  • Keep radius consistent across related components.

Do Not

  • Do not mix multiple radius scales in the same component family.
  • Do not override radius tokens without design review.

Code Examples

.card {
  border-radius: $radius-lg;
}

.badge {
  border-radius: $Radius-100;
}
